# Health Records App

A simple Python app to track health records, calculate BMI, and view statistics.

## Features
- Add and save health records
- View all records
- Calculate BMI and health category
- View statistics (average BMI, most common category, etc.)

## Requirements
- Python 3.x
- [pydantic](https://pydantic-docs.helpmanual.io/)

## Usage
1. Install dependencies:  
   `pip install pydantic`
2. Run the app:  
   `python main.py`
